# any, all() 함수

#### [1. any 함수]

----

: **함수에 포함된 값들중 어느 하나가 참이면 True를 출력하는 함수**

```
any([True, False, False]);     # True
any(target in row for row in matrix)    # 이중 배열에서 값 존재 여부 확인
```

<br>

#### [2. all 함수]

----

: 함수에 포함된 값들 전부 참이면 True를 출력하는 함수

```
any([True, True, False]);   # False
```
