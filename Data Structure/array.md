# 배열(Array)

: 데이터를 나열하고, **각 데이터를 인덱스에 대응**하도록 구성한 데이터 구조

  => **인덱스를 통한 직접 접근이 가능**



- 장점 : 빠른 접근
- 단점 : 수정(추가/삭제)이 비교적 어려움



#### 파이썬에서의 배열

```python
data = [1,2,3,4,5]
```

```python
data = [[1,2],[3,4]]  #2 차원 배열
```

```python
for x in data:  # data 배열내 데이터들을 하나하나씩 가져옴
```



##### + 배열과 리스트의 차이

- 배열(Array)는 **크기가 변하지 않는** 리스트형 자료구조(중간 값을 지워도 빈칸으로 유지)
- 리스트는 반대로 **크기가 변하는** 자료구조
